Suppose there exists a linear map on VVV whose null space and range are both finite-dimensional. Prove that VVV is finite-dimensional.

(Note 3.22 doesn't apply here since it assumes VVV is finite-dimensional. 2.34 Also can't be used till we know VVV is finite.)

Let TTT be our linear map. Let v1,…,vmv_1,\dots,v_mv1​,…,vm​ be independent vectors in VVV such that Tv1,…,TvrTv_1,\dots,Tv_rTv1​,…,Tvr​ is a basis for range T\text{range }Trange T and vr+1,…,vmv_{r+1},\dots,v_mvr+1​,…,vm​ is a basis for null T\text{null }Tnull T.

Let v∈Vv \in Vv∈V, since Tv1,…,TvrTv_1,\dots,Tv_rTv1​,…,Tvr​ is a basis for the range we can write

Tv=a1Tv1+⋯+arTvTv = a_1Tv_1 + \dots + a_rTvTv=a1​Tv1​+⋯+ar​Tv

Subtract the right hand side and use linearity to get

T(v−a1v1−⋯−arvr)=0T(v - a_1v_1 - \dots - a_rv_r) = 0T(v−a1​v1​−⋯−ar​vr​)=0

Since vr+1,…,vmv_{r+1},\dots,v_{m}vr+1​,…,vm​ is a basis for the nullspace we can write

v−a1v1−⋯−arvr=ar+1vr+1+⋯+amvmv - a_1v_1 - \dots - a_rv_r = a_{r+1}v_{r+1} + \dots + a_mv_mv−a1​v1​−⋯−ar​vr​=ar+1​vr+1​+⋯+am​vm​

Rearranging gives

v=a1v1+⋯+amvmv = a_1v_1 + \dots + a_mv_mv=a1​v1​+⋯+am​vm​

Implying span(v1,…,vm)=V\text{span}(v_1,\dots,v_m) = Vspan(v1​,…,vm​)=V meaning VVV is finite-dimensional.
