Suppose VVV is finite-dimensional and S,T∈L(V)S,T \in \mathcal L(V)S,T∈L(V). Prove that ST=IST = IST=I if and only if TS=ITS = ITS=I.

This is a rewording of 3.69, If ST=IST = IST=I then TTT is clearly injective, which by 3.69 implies TTT is invertible. Applying T−1T^{-1}T−1 to both sides of ST=IST = IST=I shows S=T−1S = T^{-1}S=T−1 which shows TS=ITS = ITS=I as desired.

The reverse direction is exactly the same if you swap TTT and SSS.
